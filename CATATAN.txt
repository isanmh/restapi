Cara install Spring Boot di Windows :
- Pastikan Java Development Kit (JDK) sudah terinstall di komputer Anda. Anda dapat mengunduh JDK dari situs resmi Oracle atau menggunakan OpenJDK.
- Unduh dan pasang Apache Maven dari situs resminya. Setelah mengunduh,
- ekstrak file ZIP dan tambahkan direktori bin Maven ke variabel lingkungan PATH.
- Buka Command Prompt dan jalankan perintah berikut untuk memeriksa apakah Maven sudah terpasang dengan benar:
  mvn -version
## Jalankan maven :
- mvn spring-boot:run

Cara install Spring Doc OpenAPI di Spring Boot :
1. Tambahkan dependensi Spring Doc OpenAPI ke dalam file `pom.xml` proyek Spring Boot Anda. Berikut adalah contoh dependensi yang perlu ditambahkan:
```xml
2. pom.xml
<dependency>
    <groupId>org.springdoc</groupId>
    <artifactId>springdoc-openapi-ui</artifactId>
    <version>1.6.14</version>
</dependency>
3. tambah config di appllication.properties :
# spring doc openapi
springdoc.api-docs.path=/v3/api-docs
springdoc.swagger-ui.path=/swagger-ui.html
